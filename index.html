Sobre mim

Sou desenvolvedor com 4 anos de experiência na Unimed Maceió, atuando em soluções para a área da saúde. Participei da transição do sistema Sabius para o Tasy, onde atuei na migração dos dados históricos dos setores financeiro e contábil, garantindo a continuidade das operações. Atualmente, apoio a equipe de negócios na otimização de processos e ajustes de relatórios. Meu objetivo principal é expandir meus conhecimentos em desenvolvimento web e análise de dados, buscando aplicar essa nova expertise em projetos desafiadores e inovadores.

Projetos desenvolvidos

  • Sorteador de números – Ver no GitHub

    Aplicação interativa desenvolvida para Sorteio de Números diretamente no navegador. O usuário poderá escolher quantos números deseja sortear, definir um intervalo mínimo e máximo (range) e visualizar o resultado em tempo real. Além disso, será possível ativar a opção para não repetir números no sorteio. O foco está em construir uma interface clara e funcional, adaptada para desktop e mobile, aplicando boas práticas de HTML, CSS e JavaScript para manipular os números de forma dinâmica.

  • Converter – Ver no GitHub

    Aplicativo desenvolvido para consolidar conhecimentos fundamentais em JavaScript, explorando conceitos de manipulação do DOM, funções e eventos. O objetivo do app é permitir que o usuário:

    • Informe um valor a ser convertido;
    • Selecione a moeda desejada;
    • Receba o resultado de acordo com uma cotação fictícia, simulando o funcionamento real de um conversor de moedas.
  • Snitap - Patins – Ver no GitHub

    Interface desenvolvida a partir de um layout de patins criado no Figma, com foco em aprimorar habilidades em animações, transições e transformações utilizando CSS puro. Este projeto foi pensado para proporcionar uma experiência visual fluida e envolvente, explorando recursos modernos do CSS aplicados a elementos interativos.

  • Zingen – Ver no GitHub

    Projeto desenvolvido a partir de um layout criado no Figma, com foco em aprimorar habilidades em responsividade utilizando CSS puro. O projeto foi pensado para garantir adaptação fluida entre diferentes dispositivos, explorando boas práticas de layout flexível e recursos modernos do CSS.

  • DevFinance – Ver no GitHub

    Projeto desenvolvido com foco em práticas modernas de front-end, utilizando HTML5, CSS3 e JavaScript puro. A aplicação permite registrar e visualizar entradas e saídas financeiras, com persistência de dados por meio do LocalStorage, simulando o comportamento de armazenamento local de uma aplicação real.
